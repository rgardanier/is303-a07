# is303-a07
seventh assignment for IS 303


I would like to describe the project to someone who may not know what it does...

This project grabs data from a URL and analyzes it, puts it into a database and then gives visuals so that someone could better understand the data that is stored in the URL.

to get a little more specific, we are scraping a website that has a bunch lf different books. It goes through the first three pages and grabs every book on the first three pages. It will grab the book name, price, and rating. It then gives analytics about the URL and the books. It will give the total amount of books, the average price among those books, most/least expensive book, and the average rating. It then creates an awesome bar graph that plots the data and gives the user a visual to better understand the data that is at hand. 

That is the project, and it can be useful in other cases too! It doesn't have to just be books that we are scraping, it can be a plethora of other categories. We would just have to change a little of the code to cater it to other websites and categories, but that change won't take that long. This code can be very useful in a corporate world when trying to analyze data on a certain website. Especially the coporates own website. Finding and anaylizing data has never been easier!
